# NLP与CV基础

## 自然语言处理（NLP）

### 文本预处理
- **分词（Tokenization）**：将文本切分为最小处理单元。中文分词常用jieba等工具，英文可用空格或子词分词
- **词性标注（POS Tagging）**：标注每个词的词性（名词、动词等）
- **命名实体识别（NER）**：识别文本中的人名、地名、组织名等实体
- **依存句法分析**：分析句子中词语之间的语法依存关系

### NLP经典任务
- **文本分类**：情感分析、主题分类、垃圾邮件检测
- **序列标注**：NER、分词、词性标注
- **文本生成**：机器翻译、文本摘要、对话生成
- **信息抽取**：关系抽取、事件抽取
- **语义匹配**：文本相似度、问答匹配、自然语言推理

### 预训练模型演进
- **BERT**：双向Encoder，通过MLM和NSP预训练，擅长理解任务
- **RoBERTa**：BERT的优化版，更大的数据、更大的Batch、更久的训练
- **ALBERT**：通过参数共享和分解嵌入减少参数量
- **DeBERTa**：引入解耦注意力机制和增强的掩码解码器
- **XLNet**：排列语言模型，结合自回归和自编码的优点
- **T5**：统一的Text-to-Text框架，将所有NLP任务转换为文本到文本的形式

## 计算机视觉（CV）

### 基础概念
- **像素（Pixel）**：图像的基本单元
- **色彩空间**：RGB（红绿蓝）、HSV（色调饱和度亮度）
- **图像分辨率**：图像的宽度×高度（像素数）
- **通道数**：灰度图1通道，彩色RGB图3通道

### CNN视觉特征提取
- 浅层卷积学习边缘、纹理等低级特征
- 中层卷积学习形状、轮廓等中级特征
- 深层卷积学习物体部件、语义等高级特征

### 经典视觉任务
- **图像分类**：判断图像属于哪个类别（猫/狗/车等）
- **目标检测**：定位并识别图像中的多个物体，输出边界框+类别
- **语义分割**：为每个像素分配类别标签
- **实例分割**：区分同一类别的不同实例
- **图像生成**：GAN和扩散模型生成逼真图像

### 目标检测算法演进
- **Two-stage方法**：R-CNN → Fast R-CNN → Faster R-CNN（先提取候选区域再分类）
- **One-stage方法**：YOLO系列、SSD（直接回归边界框和类别）
- **Anchor-free方法**：CenterNet、FCOS（不使用预设锚框）
- **Transformer检测器**：DETR（将检测视为集合预测问题）

### ViT（Vision Transformer）
将图像分割为16×16的Patch，将每个Patch视为NLP中的Token输入Transformer处理：
- 纯注意力机制替代卷积
- 需要大规模数据预训练（或使用蒸馏）
- 在大规模数据上超越CNN

### 扩散模型（Diffusion Models）
- 前向过程：逐步向图像添加高斯噪声
- 反向过程：学习逐步去噪恢复原图
- 代表模型：Stable Diffusion、DALL-E 2/3、Midjourney
- 应用：文生图、图生图、图像编辑、超分辨率

## 多模态AI

### CLIP（Contrastive Language-Image Pre-training）
OpenAI提出的图文对齐模型，使用对比学习将图像和文本映射到同一向量空间：
```
图像特征和文本描述在向量空间中距离越近，表示语义越匹配
```

### 多模态大模型
- **GPT-4V/4o**：OpenAI的多模态模型，支持图文理解和生成
- **LLaVA**：将视觉编码器与LLM连接的开源多模态模型
- **Qwen-VL**：阿里开源的多模态模型系列
- **Gemini**：Google的原生多模态模型

### 多模态应用
- 图文问答：根据图片回答自然语言问题
- 视频理解：分析视频内容生成描述或回答
- 视觉推理：结合图像进行逻辑推理
- 跨模态检索：用文本搜索图像或用图像搜索文本
