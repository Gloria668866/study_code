## 关键事实速览（30 秒版）
- **项目**：车市镜 / EV-MarketLens —— 新能源汽车市场情报 **对话式 Agent**。
- **目标**：作品级产品，主要服务 **2026 秋招拿高薪 / 未来创业**（核心卖点 = 大模型应用/Agent 工程能力）。
- **架构**：通用「双脑」引擎 + 可插拔领域包。双脑 = **Text2SQL**（查结构化数据→图表+洞察）+ **RAG**（查用户上传知识库），**LangGraph** 做意图路由。
- **数据**：爬**懂车帝销量榜 API**（公开、免登录），主源已锁定；schema 已定死见 03。
- **关键技术栈**：Scrapling 爬虫 · MinerU 解析 PDF · DeepSeek-V4 对话 · BGE-large-zh 向量 · PostgreSQL+pgvector · MinIO · FastAPI(SSE) · 单台云 VPS 部署。
- **团队**：5 人小公司（用户 + AI 是老板），**全栈制不分专职**，按工单认领。
